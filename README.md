# 시장 감지 및 진단 대시보드

## 1. 프로젝트 개요

본 프로젝트는 과거 일별 주가 데이터(OHLCV)를 활용하여 주식 시장의 상태를 입체적으로 진단하는 시스템을 구축하는 것을 목표로 합니다. 수익률, 변동성, 추세 강도 등 다양한 기술적 파생 지표를 직접 설계하고 구현하여, 500개 주요 상장 기업에 대한 다차원 분석을 제공합니다.

단순한 가격 추적을 넘어, 고도화된 변수 설계와 인터랙티브 시각화를 통해 투자 의사결정에 실질적인 도움을 주는 인사이트를 도출하는 데 중점을 둡니다.

### 핵심 목표

- **파생 변수 설계** : 원본 데이터로부터 RSI, MACD, MDD, ATR 등의 금융 파생 지표를 직접 구현합니다.
- **통계적 보정**: 표준화(Z-Score) 및 수익률 기반 지표를 사용하여, 규모가 다른 500개 종목을 동일 선상에서 비교 분석합니다.
- **Top-Down 분석 구조**: 거시적 시장 흐름(Macro)에서 시작하여 섹터(Sector)를 거쳐 개별 종목(Individual Stock)으로 이어지는 계층적 분석을 수행합니다.

## 2. 아키텍처 및 워크플로우

### 데이터 파이프라인

1. **데이터 수집**: Yahoo Finance 를 통해 최근 5년치 일별 OHLCV 데이터를 수집합니다.
2. **전처리**:
   - 결측치 및 이상치 처리.
   - 액면분할, 배당 등이 반영된 수정 주가 처리.
   - 분석을 위한 섹터 및 산업 정보 매핑.
3. **지표 산출**:
   - **추세**: 이동평균선(MA 5, 20, 60), MACD.
   - **모멘텀**: RSI, Stochastic, 거래량 급증(Volume Spikes).
   - **리스크**: 20일 변동성, 최대 낙폭(MDD), ATR.
4. **내보내기**: 시각화 도구(Tableau)에 최적화된 형태의 CSV 데이터셋 생성.

### 기술 스택

- **언어**: Python 3.12+
- **데이터 처리**: Pandas, NumPy
- **금융 라이브러리**: yfinance, ta (Technical Analysis Library)
- **시각화**: Tableau, Matplotlib

## 3. 대시보드 구조

Tableau를 활용하여 다음과 같은 드릴다운 구조로 시각화합니다.

1. **시장 조망**

   - 일간 수익률, 신고가/신저가 종목 수, 거래량 급증 종목 등을 통해 시장의 전반적인 활력을 요약합니다.

2. **섹터 히트맵**

   - 섹터별 등락률과 시가총액 비중을 트리맵으로 표현합니다.
   - 자금이 어느 섹터로 이동하고 있는지를 직관적으로 파악할 수 있습니다.

3. **종목 상세**

   - 선택한 종목의 기술적 차트와 핵심 지표를 상세 조회합니다.
   - RSI 기준값 등을 사용자가 직접 조정하여 매매 신호를 시뮬레이션할 수 있습니다.

## 4. 리포지토리 구조

- **01_Data_Loader_QA.ipynb**: 초기 데이터 로드, 정제 및 품질 검증.
- **02_Sector_Update.ipynb**: 외부 API를 활용한 누락된 섹터/산업 정보 보완.
- **03_Feature_Engineering.ipynb**: 고급 기술적 지표 산출 및 파생 변수 생성 (예정).
- **Data_set/**: 원본 및 전처리된 CSV 데이터 파일 저장소.
- **Docs/**: 데이터 사전 및 프로젝트 규칙 등 문서 자료.

## 5. 컨벤션 및 규칙

- **데이터 보존**: 모든 중간 및 최종 데이터는 반드시 CSV 포맷(utf-8-sig 인코딩)으로 저장합니다.
- **코드 품질**: 하나의 파일은 하나의 주제만 다루도록 모듈화하며, 불필요한 출력은 지양합니다.
